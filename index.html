Підтягнуто автоматично — можна редагувати
1Класифікація

Основа, за якою клієнт відфільтровує товар.

2Фізичні параметри лоту

Дають клієнту порахувати собівартість одиниці — головний аргумент при покупці.

3Наповнення лоту

Що саме людина отримає, відкривши мішок. Дивіться весь лот і відповідайте якомога конкретніше.

4Розміри

Перерахуйте одиниці по розмірних групах. Система сама порахує відсоток від загальної кількості — поле «%» можна редагувати.

Розподіл за розмірами — кількість, шт
5Тип виробу, фасони та кольори

Оберіть тип виробу — зʼявляться конкретні запитання саме для нього. Вводьте кількість, відсоток рахується сам.

Колірна гама — кількість, шт
Фасон за малюнком
Наявність коміра

Уся позиція — фліс. Матеріал не питаємо. Вводьте кількість одиниць по кожній ознаці, % рахується сам.

6Стан та якість

Найважливіше для секонду — наскільки товар свіжий і в якому він стані.

Новизна — розподіл одиниць — кількість, % сам
7Дефекти

Кожен вид дефекту — окреме поле. Вводьте кількість бракованих одиниць, відсоток від загальної кількості рахується сам (можна редагувати).

8Деталі по категорії

Зʼявляться запитання, специфічні для обраної категорії.

Вводьте кількість пар — відсоток рахується автоматично. Поле «%» можна редагувати.
